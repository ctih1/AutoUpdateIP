# [AutoUpdateIP](https://github.com/ctih1/AutoUpdateIP)
## Automatically updates your Cloudflare domain's IP.

# Installation

1. Drag the .jar file into /plugins/
2. Fill in the fields in plugins/AutoUpdateIP/config.yml
3. Start the server.
